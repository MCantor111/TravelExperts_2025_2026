Important Notes ::

In packages, you must enter the credentials of a valid registered user to make a booking

The message functionality in contacts is merely placeholder,
and does not actually post its contents anywhere

Instructions ::

1) Have the TravelExperts MySQL database running on your machine
2) Open a terminal in the phase1-server project folder, and type 'npm run dev'
3) Go to the URL specified in the terminal
4) Go to the register page, and fill out the form correctly
A message should display saying that you have successfully registered
5) Check the MySQL TravelExperts database
The form contents should now be successfully posted
